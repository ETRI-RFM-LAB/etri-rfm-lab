ABOUT ETRI RFM LAB

스스로 배우고 행동하는
차세대 로봇 지능

ETRI 로봇파운데이션모델연구실은 대규모 멀티모달 데이터를 기반으로 인지·이해·행동을 통합하는 로봇 파운데이션 모델을 개발합니다.

  • 멀티모달 Robot Foundation Model
  • Vision-Language-Action 기반 행동 생성
  • 실환경 경험 기반 자율학습 파이프라인
  • 휴머노이드 로봇 시스템 적용 및 검증
CORE RESEARCH

Robot Foundation Model

범용 로봇 지능의 기반 모델부터 실제 휴머노이드 시스템까지 통합적으로 연구합니다.

RFM

Multimodal Foundation Model

시각·언어·로봇 상태·행동 데이터를 하나의 표현 공간에서 학습합니다.

VLA

Vision-Language-Action

자연어 지시와 장면을 이해하여 로봇이 실행할 행동을 생성합니다.

AL

Autonomous Learning

실환경 경험으로 성능을 지속적으로 개선하는 자율학습 파이프라인을 연구합니다.
